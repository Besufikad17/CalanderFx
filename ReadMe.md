<h1>CalanderFx</h1>

CalanderFx is javaFx app which used as a calander based on Ethiopic calander system.

## Prerequisites

Before you begin, ensure you have met the following requirements:

* ```Abushakir.jar```
* ```guava-19.0.rcl.jar```
